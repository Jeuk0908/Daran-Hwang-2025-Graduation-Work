# 졸업전시 프론트엔드 프로젝트

## 📱 프로젝트 소개
졸업 전시를 위한 웹앱 프론트엔드 프로젝트입니다. 실제 백엔드 연결 없이 전시/시연 목적으로 제작되었습니다.

## 🚀 시작하기

### 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```

### 빌드
```bash
npm run build
```

### 프리뷰
```bash
npm run preview
```

## 🛠 기술 스택
- React 19.1.1
- Vite 7.1.7
- React Router DOM
- JavaScript (JSX)

## 📁 프로젝트 구조
```
src/
├── pages/              # 페이지 컴포넌트
├── components/         # 재사용 컴포넌트
│   ├── common/        # 공통 컴포넌트
│   ├── layout/        # 레이아웃
│   └── features/      # 기능별 컴포넌트
├── hooks/             # 커스텀 훅
├── services/          # Mock 데이터
├── store/             # 상태 관리
├── utils/             # 유틸리티
├── styles/            # 스타일
└── assets/            # 정적 파일
```
